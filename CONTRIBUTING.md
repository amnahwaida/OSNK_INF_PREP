# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada **OSNK_INF_PREP**! Kami sangat menghargai bantuan Anda dalam meningkatkan kualitas materi persiapan OSN-K Informatika ini.

## Cara Berkontribusi

### 1. Temukan Typo atau Kesalahan Materi
Jika Anda menemukan kesalahan penulisan, link yang mati, atau kesalahan pada pembahasan soal:
- Buka **Issue** baru dan jelaskan kesalahannya.
- Atau, lakukan **Pull Request (PR)** dengan perbaikan langsung.

### 2. Tambahkan Soal Latihan Baru
Kami selalu mencari soal-soal baru yang berkualitas:
- Gunakan folder `assets/` untuk gambar pendukung.
- Ikuti format Markdown yang sudah ada (menggunakan `<details><summary>` untuk jawaban).
- Pastikan soal relevan dengan tipe OSN-K (Part A, B, atau C).

### 3. Perbaiki Dokumentasi
Membantu merapikan `README.md` atau panduan strategi juga sangat kami apresiasi.

## Aturan Komunitas
- Gunakan bahasa Indonesia yang baik dan sopan.
- Pastikan materi yang Anda tambahkan tidak melanggar hak cipta (disarankan materi original atau adaptasi dengan atribusi).
- Karya yang Anda kontribusikan akan berada di bawah lisensi **CC BY-SA 4.0**.

## Proses Pull Request
1. Fork repositori ini.
2. Buat branch baru untuk fitur/perbaikan Anda (`git checkout -b fitur-baru`).
3. Commit perubahan Anda (`git commit -m 'Menambahkan soal logika Part A'`).
4. Push ke branch tersebut (`git push origin fitur-baru`).
5. Buka Pull Request.
