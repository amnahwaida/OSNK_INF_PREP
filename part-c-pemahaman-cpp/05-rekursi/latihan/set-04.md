```

1. Berapakah hasil akhirnya?

```mermaid

Lantai_3 --> Lantai_2 --> Lantai_1 --> Lantai_0
```

**📖 Penjelasan Komprehensif:**

2. **Call Stack**: Mesin mengingat setiap lantai yang dilewati (n=3, n=2...).
