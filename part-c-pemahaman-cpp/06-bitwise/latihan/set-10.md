```cpp

```
**Pertanyaan:**
1. Berapakah hasil akhirnya?
2. Mengapa demikian?

**Jawaban & Diagnosis:**
1. **112**
2. Lihat Tracing.

**Mermaid Flowchart:**
```mermaid

B --> C["112"]
```

**📖 Penjelasan:**

2. Jalankan <<.
