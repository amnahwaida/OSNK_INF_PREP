# Latihan Soal Part C - Modul 06 - Set 07

### Soal 151
```cpp
int res = 15 & 7;
```
**Pertanyaan:**
1. Berapakah hasil akhirnya?
2. Mengapa demikian?

**Jawaban & Diagnosis:**
1. **7**
2. Lihat Tracing.

**Mermaid Flowchart:**
```mermaid
graph LR
A["15 (0b1111)"] --> C["&"]
B["7 (0b111)"] --> C
C --> D["7"]
```

**📖 Penjelasan:**
**Langkah Tracing:**
1. Ubah 15 ke biner (0b1111).
2. Jalankan &.
3. Hasil: 7.
