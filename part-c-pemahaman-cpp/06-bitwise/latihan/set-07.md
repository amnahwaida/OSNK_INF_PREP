```cpp
// Ohm: Shift Left
int val = 3;
int res = val << 1;
```
**Pertanyaan:**
1. Berapakah hasil akhirnya?
2. Deskripsikan alur pikir 'Compiler Manusia' untuk soal ini!

**Jawaban & Diagnosis:**
1. **6**
2. 3 digeser kiri 1x = dikali 2 = 6.
