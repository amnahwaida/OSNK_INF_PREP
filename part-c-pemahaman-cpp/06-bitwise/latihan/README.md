### 📖 Panduan Membaca Diagram Bitwise
Operasi bitwise bekerja di level biner (0 dan 1). Untuk memudahkan, kami menyediakan **Bit-Grid Diagram**.

1.  **Kotak 8-Bit**: Menunjukkan posisi bit dari bit ke-7 (paling kiri) hingga bit ke-0 (paling kanan).
2.  **Operator Symbols**:
    - `&` (AND): Kedua bit harus 1 untuk menghasilkan 1.
    - `|` (OR): Salah satu bit 1 sudah cukup untuk menghasilkan 1.
    - `^` (XOR): Bit harus berbeda untuk menghasilkan 1 (Manusia vs Alien).
    - `~` (NOT): Membalik semua bit.
    - `<<` / `>>`: Menggeser seluruh barisan bit ke kiri atau kanan.
