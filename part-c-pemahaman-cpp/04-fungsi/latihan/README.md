### 🧩 Panduan Membaca Diagram Mermaid (Call Stack)
Untuk membantu kamu memahami alur "lompatan" kode antar fungsi, sebagian besar soal dilengkapi dengan diagram **Mermaid**. Berikut cara membacanya:
- **Kotak Persegi `[...]`**: Menandakan proses atau deklarasi variabel (Contoh: `x = 10`).
- **Kotak Belah Ketupat `{...}`**: Menandakan percabangan atau kondisi `if` (Contoh: `x > 5?`).
- **Panah `-->`**: Arah eksekusi baris kode (dari pemanggil ke fungsi).
- **Label `Alias (&)`**: Menandakan pengiriman parameter dengan referensi (perubahan di dalam berakibat ke luar).
