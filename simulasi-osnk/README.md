Selamat datang di area gladi bersih! 🏁

## ⚖️ Aturan Pengerjaan Simulasi:
1. **Siapkan Timer:** Setel alarm selama 150 menit (2 jam 30 menit).
2. **Kertas Buram & Pensil:** Siapkan setidaknya 5 lembar kertas HVS kosong. Kamu wajib membuat diagram, tabel *dry run*, dan memecah tumpukan rekursi secara manual.
3. **No Calculator / No Compiler:** Jangan buka IDE C++, Visual Studio Code, replit, atau kalkulator hp. Biasakan jarimu menghitung modulo besar dan XOR!
4. **Strategi Menjawab:** 
   - Kerjakan soal yang mudah dulu (biasanya algoritma pola Part A, atau percabangan sederhana).
   - Simpan soal DP 2D atau rekursi maut (misal: Ackermann function) di bagian akhir waktu.
   - Pahami aturan pemotongan skor jika ada (Benar +4, Salah -1, Kosong 0).

Ayo ukur kemampuanmu dan rasakan atmosfer sesungguhnya dari Olimpiade! 🎉
